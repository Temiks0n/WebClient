КЛИЕНТСКАЯ ВЕБ-РАЗРАБОТКА (HTML, CSS, JAVASCRIPT)

Старт: Идет набор

Цена: 12000 руб. 13000 руб. (Возможная оалата по частям)

Цена для студентов: 9000 руб.

Содержания курса:

В курсе веб-разработке мы познакомим вас с клиентской веб-разработкой, которая включает в себя верстку - обучения HTML и CSS, и написания кода на JavaScript. На курсе JavaScript будут рассмаотрены синтаксис, типы данных, работа с DOM и событиями, стандартная библиотека языка.

В курсе JavaScript также рассматривается большое количество библиотек и средств разработки jQuery, jQuery UI, Underscore, Vue.js, Bootstrap, npm, Webpack, Less/Sass, Babel. Также оброзно рассмотрим Grunt, RequireJS, TypeScript. В качестве серверной части используем Express.js. Обучения веб-разработки закрепляется на практике.

Вы научитесь создавать клиентскую часть сайта и получите кругозор по клиентским технологиям.
